💰 What is Billing System?

Billing System is used to create client bills. It calculates services, GST, discount, advance and balance. Finally, we can download PDF or send bill on WhatsApp.

🧾 How to create a new bill?

Click "New Bill" button to start fresh bill. Enter client details and select required services. System will automatically calculate all amounts.

🧑 How to add client details?

Fill client name, phone, address and GST number. All details are saved with the bill for future reference.

🛒 How to select services?

Choose services from dropdown list. Each service has price and GST rate. Multiple services can be added to same bill.

🧮 How GST is calculated?

System calculates GST automatically based on service rates. GST percentage is predefined for each service category.

🎁 How discount works?

Enter discount percentage or amount. System will reduce total bill amount and recalculate balance accordingly.

💵 What is advance paid?

Advance is amount paid by client before final bill. Remaining balance is calculated after deducting advance from total.

📌 What is balance amount?

Balance is remaining amount after deducting advance payment from total bill amount. Client needs to pay this balance.

👀 What is Check Invoice?

Check Invoice shows preview of final bill before generating. Use this to verify all details are correct.

🔒 What is Generate Invoice?

Generate Invoice finalizes the bill and assigns unique invoice number. Bill cannot be modified after generation.

📄 How Download PDF works?

Download PDF creates professional invoice document. PDF can be printed or shared with client.

📲 How Send on WhatsApp works?

Send on WhatsApp opens WhatsApp with pre-filled invoice message and PDF attachment.

🖨️ How Print Bill works?

Print Bill opens print dialog to print invoice directly on printer.

💾 How Save Bill works?

Save Bill stores bill in browser LocalStorage for future access. Bills remain saved even after refresh.

🗂️ What is View History?

View History shows all previously generated bills. Can search and download any past invoice.

🧹 What does Clear Form do?

Clear Form removes all entered data and starts fresh bill creation.

🔢 How invoice number works?

Invoice number is auto-generated in format INV-YYYY-XXX where XXX is sequential number.

💽 How LocalStorage saves bills?

LocalStorage saves billing data inside browser memory. Bills remain saved after refresh and laptop restart. No internet database is needed currently.

⚠️ What happens if browser data is cleared?

If browser cache or browser data is cleared, saved bills may be deleted permanently.

📤 Why Export CSV is important?

Export CSV creates backup of billing records. If system/browser data is lost, CSV backup can recover important client bills.

🛡️ Why backup is important?

Backup protects company billing history. Always export CSV regularly for safety.

⚠️ Important Data Safety Note

✅ LocalStorage keeps data after restart

✅ Export CSV regularly for backup

⚠️ Clearing browser data may remove saved records

💾 Always keep backup files safe

📞 What is CRM?

CRM (Customer Relationship Management) manages client leads and business contacts. Helps track potential customers and convert them into clients.

📊 What is dashboard?

Dashboard shows total leads, new leads, interested leads, follow-ups today and converted clients in one view.

📈 What is lead graph/chart?

Lead graph shows visual representation of lead status and conversion rates over time.

🕘 What are recent leads?

Recent leads shows latest added businesses for quick follow-up and contact.

➕ How to add a lead?

Fill business name, phone, category, services, budget and follow-up date. Click Add Lead to save.

🏢 What is business category?

Business category classifies leads like Hospital, Salon, School, Shop etc. for better organization.

🛠️ What are required services?

Services are what client needs like Websites, SEO, Ads, AI Agents, Posters etc.

📌 What is lead status?

Lead status shows progress: New Lead, Called, Interested, Not Interested, Call Later, Meeting Fixed, Converted Client.

🔍 How search works?

Search finds leads by business name or phone number from all saved leads.

🧰 How filters work?

Filters narrow leads by category, status or services for focused view.

👁️ What does View lead do?

View lead shows complete details of selected business with all information.

✏️ What does Edit lead do?

Edit lead allows modifying any lead information like status, follow-up date or notes.

📲 How WhatsApp button works?

WhatsApp button opens WhatsApp chat with client number for quick communication.

☎️ How call button works?

Call button opens dialer with client number for making phone calls.

🗓️ What are follow-ups?

Follow-ups are scheduled calls or meetings to maintain contact with potential clients.

🏷️ What are categories page?

Categories page shows all business types with lead count for each category.

⚙️ What are services page?

Services page shows all services offered with lead count for each service.

📤 What is export CSV?

Export CSV downloads all leads into Excel-style file. Useful for backup and transferring data.

📥 What is import CSV?

Import CSV allows restoring leads from backup file. Useful for data migration.

💾 How LocalStorage saves leads?

All client leads are saved inside browser LocalStorage. Leads remain after refresh and restart.

⚠️ What happens if browser data is cleared?

If browser data/cache is cleared, all saved leads may be deleted.

📤 Why Export CSV is important?

Export CSV creates backup of all client leads. Regular backup prevents data loss.

🛡️ Why lead backup is important?

Client leads are valuable business data. Regular backup prevents data loss.

⚠️ Important Data Safety Note

✅ LocalStorage keeps leads after restart

✅ Export CSV regularly for backup

⚠️ Clearing browser data may remove saved leads

💾 Always keep CSV backup files safe

🎨 What is Design & Tasks?

Design & Tasks manages creative work assignments. Assigns tasks to team members with deadlines and requirements.

📊 What is task dashboard?

Task dashboard shows total tasks, pending, working, completed and delayed tasks overview.

➕ How to add a task?

Fill task details like title, category, assigned to, deadline and priority. Click Add Task to create.

🆔 What is Task ID?

Task ID is unique number for each task like TK-001, TK-002 etc. for easy reference.

👨‍💻 What is Assigned To?

Assigned To selects team member responsible for completing the task.

📂 What is task category?

Task category classifies work like Website Design, Logo Design, Video Editing, SEO Support etc.

📝 What is task title?

Task title describes the work to be done in short, clear description.

⏰ What is deadline?

Deadline is due date for task completion. Tasks can be marked delayed if not completed on time.

🚦 What is task status?

Task status shows progress: Pending, Working, Review, Completed, Delayed.

🔥 What is priority?

Priority sets task importance: Low, Medium, High, Urgent. Higher priority tasks get attention first.

🔗 What are work links?

Work links provide access to related files, code repositories and project resources.

🐙 What is GitHub link?

GitHub link opens code repository for website development projects.

📁 What is Google Drive link?

Google Drive link opens file storage for design files and documents.

🖼️ What is screenshot link?

Screenshot link shows visual reference or preview of design work.

👁️ What does View task do?

View task shows complete task details with all information and links.

✏️ What does Edit task do?

Edit task allows modifying any task details like status, deadline or links.

▶️ What does Mark Working do?

Mark Working changes task status to "Working" when team member starts work.

✅ What does Mark Completed do?

Mark Completed changes task status to "Completed" when work is finished.

📲 How WhatsApp Worker works?

WhatsApp Worker opens chat with assigned team member for task coordination.

📤 What is Export CSV?

Export CSV creates backup of all tasks and assignments in Excel format.

📥 What is Import CSV?

Import CSV restores task data from backup file for data recovery.

💾 How LocalStorage saves tasks?

All tasks are stored in browser LocalStorage. Tasks remain after refresh and restart.

⚠️ What happens if browser data is cleared?

If browser data is removed, saved tasks may disappear permanently.

📤 Why Export CSV is important?

Export CSV creates backup of tasks and assignments. Regular backup protects project history.

🛡️ Why task backup is important?

Backup helps protect project history and assigned works. Prevents data loss.

⚠️ Important Data Safety Note

✅ LocalStorage keeps tasks after restart

✅ Export CSV regularly for backup

⚠️ Clearing browser data may remove saved tasks

💾 Always keep CSV backup files safe
